# 基于[scrcpy](https://scrcpyapp.org/)和[sndcpy](https://github.com/rom1v/sndcpy)的手机投屏工具 *Rinku Sutato*

## 一.简介

1.Rinku Sutato是一款**高帧率低延迟**(60hz,30ms)的手机投屏工具,非常适合作者这种想玩原伸电脑空间不足,只能下载手机版而手机屏幕又损坏影响视野的离谱人士将手游投屏到电脑上游玩.
2.s5k1dayo编写了自带参数的scrcpy和sndcpy启动和检测脚本,将Rinku Sutato.exe快捷方式放到桌面即可一键启动投屏

## 二.使用方式

1.release中保留了scrcpy的启动程序**scrcpy.exe**首次使用本工具时应使用此exe测试usb及手机设置的正确性.

2.确认从**scrcpy.exe**能够启动成功投屏,以后可直接打开**Rinku Sutato**工具使用.

3.一般情况下scrcpy可以传声,如果按下**启动!**键后没有声音,可额外按**声音**键启动sndcpy工具.

4.sndcpy运行会自动向手机安装所需的.apk,部分手机会报告为不安全,无视风险继续安装即可.

## 三.说明

1.本程序为集成的scrcpy和sndcpy启动和监测脚本,关于二者的详细信息参见官网[scrcpy](https://scrcpyapp.org/),[sndcpy](https://github.com/rom1v/sndcpy).

2.请务必解压到非中文路径下运行.
